# simple-voice

A simple voice activity detection utility.

## Setup

1. clone the repository
```shell
git clone https://github.com/ak314/simple-voice.git
cd simple-voice
```
2. install dependencies
```shell
uv sync
```
3. (if needed) record test file
```shell
uv run record_test_file.py
```
4. test vad
```shell
uv run try_vad.py
```
